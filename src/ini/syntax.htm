Syntaxe des fichiers ini

Commentaires
; Les commentaires sont préfixés par le symbole ";".
Affectation
variable = value La variable variable prend pour valeur la valeur value.
Mots clés et variable
Mots clés
null
yes
no
true
false
on
off
none
 Les mots clés ne doivent pas être utilisés comme nom de variable.

Utilisé des nom de variables simples :

  1. Composé avec des caractères de l'alphabet et des nombres.
  2. Chaque mot séparé avec le symbole '_'.
Groupement
[section_name] Les variables suivantes sont regrouper dans la section section_name.
Valeurs
Une variable est affectée à une ou plusieurs valeurs à la suite.
mot Attention: Si des caractères sont non autorisé utiliser les chaînes.
