

Un automate possède :
	- I : un état initial
	- F : un ou plusieurs états finaux
	- A : un alphabet
	- S : des états
	- T : des transitions
	- Une liste de nom de token pour les états finaux.
	
I: Etat initial
	Un nombre
	
F: Etats finaux
	Tableau de nombre unique.

A: Alphabet
	Tableau contenant :
		- des caractères
		- des ensembles de caractère négatif ou non
		- le symbole EPSILON
		- le symbole ANY
		
S : Etats ( states en anglais)
	Tableau de nombre unique.

T: Transitions
	Implémentation la plus simple d'une transition : un tableau de 3 à 4 éléments.
		[0] un état initial
		[1] un caractère, ou un ensemble négatif ou non; de l'alphabet
		[2] un état final
		[3] une fonction permettant de transiter (cas ensemble de caractère)
				var f =function( caractere, etatFinal )

Instance(s) Automate
	buildTable		
		Construit la matrice M de l'automate
		Pour les DFAs:
			+ préparation à la minimisation de l'AFD
			+ possibilité de teste de l'automate (AFD)
	clone
		Duplique un automate en renommant les états pour que leur nom soit unique.
	renameStates
		Renomme les états de l'automate
			Note: les états puits sont égale à '0' ou 0 au départ
			arguments ([ nStateIDCounter [, bAll [, aOrder ]]])
				par défaut ( 0: état puit, 1: état initial , les autres états commencent à 2 )
				sinon si bAll==false ils commencent à nStateIDCounter sauf ( 0: état puit, 1: état initial )
				sinon si bAll==true ils commencent à nStateIDCounter 
				si ! aOrder
					calcul le poids? et organise les états en fonction
					sinon respect l'ordre des états aOrder

	
Objet Automate

	Fonctions spéciales
		action ( sSymbols, bNegated ) -> Function
			...
		getUniqueID -> Number
			retourne un nombre unique (état)
		setUniqueID -> void
			utilisé dans aggrégation d'AFD ?
		wrapper ( s ) -> Automate
			créé un automate depuis une chaine
				Choix possible
					CARACTERES SPECIAUX: '\\n', '\\t', '\\f', '\\r', '\\v'
					ENSEMBLE DE CARACTERES: '\\d', '\\D', '\\s', '\\S', '\\w', '\\W'
				Si chaine.length = 1 : automate à une transition (à un caractère)

	Fonctions retournant une fonction générant un automate unique
		fromChar ( s ) -> Function
			Automate contenant une seule transition (à un caractère).
		fromCharClass ( aSet, bNegated ) -> Function
			Automate contenant une seule transition (à un ensemble de caractère négatif ou non).

	Fonctions générant un automate unique
		Elles ont pour nom :
			CARACTERES SPECIAUX: '\\n', '\\t', '\\f', '\\r', '\\v'
			ENSEMBLE DE CARACTERES: '\\d', '\\D', '\\s', '\\S', '\\w', '\\W'
		Elles n'ont pas d'argument.
